Say Hello-World! To Docker

官方文档在这里Welcome to the Docs - Docker

安装Docker

Ubuntu 14.04: 

# install docker-engine
sudo apt-get update
sudo apt-get install apt-transport-https ca-certificates
sudo apt-key adv --keyserver hkp://ha.pool.sks-keyservers.net:80 --recv-keys 58118E89F3A912897C070ADBF76221572C52609D

sudo echo "deb https://apt.dockerproject.org/repo ubuntu-trusty main" > /etc/apt/source.list.d/docker.list

sudo apt-get update
sudo apt-get install docker-engine

按照官方文档的教程来运行命令docker run hello-world，发现报错：

docker: Cannot connect to the Docker daemon. Is the docker daemon running on this host?.

经过查询是权限的问题，切换到root用户sudo su，再次运行以上命令即可
