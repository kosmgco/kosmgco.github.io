Ubuntu 下访问Windows 硬盘

环境:

操作系统: Ubuntu 14.04

硬盘: 三星固态硬盘120G

操作系统: Windows 10

硬盘: 1T机械硬盘

平常使用Ubuntu系统，Windows系统下的硬盘作为备份硬盘，但是经常会出现使用windows后，在Ubuntu系统下不能使用Windows分区的硬盘，提示

使用命令:

mount -o ro /dev/sda6 /media/user/
# /dev/sda6是根据提示信息进行修改，可以为sda5....

意思是将/dev/sda6挂载到/media/user/
