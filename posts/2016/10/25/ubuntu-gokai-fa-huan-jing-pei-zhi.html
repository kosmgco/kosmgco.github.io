Ubuntu-Go开发环境配置

Ubuntu Go

下载Go包

wget -c "https://storage.googleapis.com/golang/go1.7.3.linux-amd64.tar.gz"

解压

tar -zxvf go1.7.3.linux-amd64.tar.gz
sudo mv go/ /opt/local/

配置GOROOT/GOPATH

# vim ~/.profile
export PATH=$PATH:/opt/local/go/bin

# vim ~/.bashrc
export GOROOT="/opt/local/go"
export PATH="$PATH:$GOROOT/bin"
export GOPATH="/home/user/GoProjects/Golang"

Go Jetbrains 插件

File > Settings > Plugins
search Go
install ...

File > Settings > Language & Frameworks > Go > Go SDK
set path=/opt/local/go

Ubuntu查找大文件

sudo find / -type f -size +800M -print0 | xargs -0 du -h

查找/路径下的 类型为文件的 大小大于800M的文件然后显示出来

定时更换ubuntu系统桌面背景

#!/usr/bin/env python
# coding:utf-8
# bing.py

import re
import os
import requests
import datetime

req = requests.Session()

setting_to_background_cmd = 'gsettings set org.gnome.desktop.background draw-background false && gsettings set org.gnome.desktop.background picture-uri file:///home/scdev/images/{date}/{filename} && gsettings set org.gnome.desktop.background draw-background true'

req.headers = {
    'User-Agent': 'Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/53.0.2785.116 Safari/537.36',
    'Accept': 'text/html,application/xhtml+xml,application/xml;q=0.9,image/webp,*/*;q=0.8'
}
url = 'http://www.bing.com/HPImageArchive.aspx?format=js&idx=0&n=1&nc=1409879295618&pid=hp'

url = req.get(url).json().get('images', [])[0].get('url', '')

url = url if url.startswith('http') else 'http://www.bing.com' + url

base_path = os.path.abspath('')

if os.path.isdir(base_path + '/images/' + datetime.datetime.today().strftime('%Y/%m/%d/')):
    pass
else:
    os.system('mkdir -p ' + base_path + '/images/' + datetime.datetime.today().strftime('%Y/%m/%d/'))

with open(base_path + '/images/' + datetime.datetime.today().strftime('%Y/%m/%d/')  +url.split('/')[-1], 'wb') as buf:
    buf.write(req.get(url).content)

os.system(setting_to_background_cmd.format(
    date = datetime.datetime.today().strftime('%Y/%m/%d'),
    filename = url.split('/')[-1]
))

环境:

操作系统: Ubuntu 14.04

硬盘: 三星固态硬盘120G

操作系统: Windows 10

硬盘: 1T机械硬盘

平常使用Ubuntu系统，Windows系统下的硬盘作为备份硬盘，但是经常会出现使用windows后，在Ubuntu系统下不能使用Windows分区的硬盘，提示

使用命令:

mount -o ro /dev/sda6 /media/user/
# /dev/sda6是根据提示信息进行修改，可以为sda5....

意思是将/dev/sda6挂载到/media/user/
