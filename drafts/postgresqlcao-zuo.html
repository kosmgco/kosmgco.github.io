PostgreSql操作

安装

sudo apt-get install postgresql-client
sudo apt-get install postgresql
sudo apt-get install pgadmin3

操作

修改默认密码

sudo su postgress
psql
\password postgress

数据库备份恢复

# 备份成sql文件
# -h host
# -U user
# -p port
# -d database
# -t table
# -a data-only 只导出数据，不导出表结构
# -f output file
# --inserts 导出成insert into table values(). 没有field，所以会导致一个问题，备份表和恢复表的表结构要完全一致，数据字段的排列顺序都要一致.缺少这个参数会导出成copy table (fields) from stdin: 这样恢复比较快也比较准确.
# -n only schema 只导出表结构，不导出数据
# -W force password 强制输入密码
# -w no-password 不需要密码
pg_dump -h host -U user -p port -d database -t table -W -a -f file.sql

# 恢复
psql -h host -U user -p port -d database -W file.sql

# 创建btree索引
create index company_name_court_zhixing_index on court_zhixing using btree (company_name);

使用 --inserts 和不使用这个参数恢复的速度是千差万别的，所以尽量不要使用 --inserts 参数
